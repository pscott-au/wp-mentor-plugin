A simple plugin to manage Mentor and Mentee registration and matching.

== Description ==

This plugin was created for SunshineCoast Chamber Alliance ( https://sunshinecoastchamberalliance.com.au/ ) as part of their Mentoring program.

== Installation ==

Installing "CCP Mentor Plugin" can be done either by searching for "CCP Mentor Plugin" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
